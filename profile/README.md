# X:AI

*X:AI는 **eXtension: Artificial Intelligence**의 약자로, 학과 수업을 넘어 인공지능 논문을 주체적으로 읽고 구현하는 학회입니다.*

다양한 인공지능 기초 지식을 학습하며, 관심 분야를 선정하여 컨퍼런스와 논문 토론 세션을 진행합니다. 딥러닝을 공부한 전공생을 대상으로 하며, 함께 공부하고 성장할 수 있는 분위기를 지향합니다.
